The Real-Time Lightning STT WebSocket API supports the following features:

## Available Features

<CardGroup cols={2}>
  <Card title="Word Timestamps" icon="clock" href="/v4.0.0/content/speech-to-text-new/features/word-timestamps">
    Get precise timing information for each word in the transcription
  </Card>
  <Card title="Language Detection" icon="language" href="/v4.0.0/content/speech-to-text-new/features/language-detection">
    Automatically detect the language of the audio
  </Card>
  <Card title="Sentence Timestamps (Utterances)" icon="comment-dots" href="/v4.0.0/content/speech-to-text-new/features/utterances">
    Get sentence-level transcription segments with timing
  </Card>
  <Card title="PII & PCI Redaction" icon="shield" href="/v4.0.0/content/speech-to-text-new/features/redaction">
    Automatically redact personally identifiable information and payment card information
  </Card>
  <Card title="Full Transcript" icon="file-text" href="/v4.0.0/content/speech-to-text-new/features/full-transcript">
    Get cumulative transcript received till now in responses where is_final is true
  </Card>
</CardGroup>
