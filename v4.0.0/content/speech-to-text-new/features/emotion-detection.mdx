## Enabling emotion detection

Include `emotion_detection=true` in your Lightning STT query parameters.

### Sample request

```bash
curl --request POST \
  --url "https://waves-api.smallest.ai/api/v1/lightning/get_text?model=lightning&language=en&emotion_detection=true" \
  --header "Authorization: Bearer $SMALLEST_API_KEY" \
  --header "Content-Type: audio/wav" \
  --data-binary "@/path/to/audio.wav"
```

<Note>
Emotion detection is currently supported only for Pre-Recorded API. Real-Time API support is coming soon.
</Note>
